HTML

Hypertext Markup Language

It is a markup language that helps you create your own webpage. It helps us communicate with others on the world wide web. Tells browsers how webpages should be structured.

CSS

Cascading Style Sheets

This helps control the look of the website. It allows you to separate the style and content. Allows you to maintain and share sites and style sheets.

Bootstrap version 4

Its is a front end framework for web development that is easy to use. It can create responsive designs. It will be compatible for any size of device
